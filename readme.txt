Fully interactive and fast loading iframe.

== Description ==
Embed youtube, vimeo or google maps using this responsive and fast loading iframe.

= Features =
* Fully Responsive and Interactive.
* Lazy Load.
* Additional Parent class.
* Custom Width and Height.
* Enable or Disable for Login users.
* Add to Widget.
* Add to Post Content(Media button).

= Shortcode Attributes =
* src ( src=" " )
* width ( width="" )
* height ( height="" )
* class ( class="" )
* login ( login="true or false" ) - If "true" then only logged in users can view the iframe. If "false" or (empty) then all users can view the iframe.

== Installation ==
= Automatic installation =
1. log in to your WordPress dashboard, go to the Plugins menu and click Add New.
2. Search for “iframe - Responsive, Lazy Load”.
3. Once found, install by clicking “Install Now” button.
4. Click on activate after installation.
5. A menu will appear under Settings main menu after activation( iframe - Responsive, Lazy Load ).

= Manual installation =
1. Download the plugin(zip format).
2. Go to Plugin dashboard, Add new and upload the zip folder.
3. Activate the plugin after unzipping/installation.
5. A menu will appear under Settings main menu after activation( iframe - Responsive, Lazy Load ).
